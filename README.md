# General Knowledge Quiz (Python)

This project is a simple command-line quiz application built in Python. It tests the user's general knowledge using conditional logic and handles answers in a case-insensitive way.
## Objective
* Ask the user 3 questions
* Evaluate answers using **if-else statements**
* Accept answers in **any case (uppercase/lowercase)**
* Assign 1 point for each correct answer
* Display the final score
## Features
* Interactive quiz system
* Case-insensitive answer checking (e.g., Paris, PARIS, paris all work)
* Removes extra spaces from input
* Score tracking
* Instant feedback
* Final performance evaluation
## Technologies Used
* Python
* Concepts:
  * Variables
  * Conditional statements (if-else)
  * User input
  * String methods (`lower()`, `strip()`)
